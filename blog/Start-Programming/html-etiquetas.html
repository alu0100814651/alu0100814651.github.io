ETIQUETAS DE HTML

Vamos a mostrar aquí el Listado de las diferentes etiquetas del HTML (y HTML5) diferenciadas según sus tipos:

Documento HTML Metadatos Agrupación Texto Multimedia Tablas HTML Formularios Scripting Sección Ideogramas Edición

Documento HTML
- Html (<html>)
- Head (<head>)
- Body (<body>)
Metadatos
- Title (<title>)
- Base (<base>)
- Link (<link>)
- Meta (<meta>)
- Style (<style>)
Agrupación
Para Agrupaciones Generales:
- Capa o División utilizado para Agrupar varias Etiquetas HTML (<div>)
- Definir un Párrafo de Texto: (<p>)
- Establece un Texto Preformateado (<pre>)
- Agrupar Información y Características de una Cita (<blockquote>)
- Contenedor para Englobar la parte Principal de la Página (<main>)
- Indicar una Separación Temática del Texto: (<hr>)
Para Agrupaciones en formato Lista:
- Definir una Lista SIN Orden (<ul>)
- Definir una Lista CON Orden (<ol>)
- Definir un Ítem de la Lista (<li>)
Para Agrupaciones en formato Lista de Descripciones:
- Definir una Lista de Descripciones (<dl>)
- Término de la Descripción (<dt>)
- Descripción o Valor Asociado al Término (<dd>)
Para Agrupaciones en Formato Ilustraciones (figuras):
- Establece una Figura, que puede contener una Serie de Elementos Diversos (<figure>)
- Asocia una Leyenda, Generalmente Texto, a la Figura Anterior (Optional) (<figcaption>)
Texto
Para Fragmentos de Texto:
- Fragmento de Texto Importante o Palabras Clave (<strong>)
- Fragmento de Texto Enfatizado Respecto a la Frase que lo Contiene (<em>)
- Fragmento de Texto Resaltado, Simulado a estar Marcado con Rotulador Amarrillo(<mark>)
- Fragmento de Texto con Voz o Tono Alternativo al Resto (<i>)
- Fragmento de Texto Sin Importancia destacable (fines utilitarios) (<b>)
- Fragmento de Texto para Nombres Propios o Escritura Incorrecta Intencionada (sic) (<u>)
- Fragmento de Texto Inexacto o qeu ya no es Relevante (errores o inexactitudes) (<s>)
- Fragmento de Texto Sin Significado (útil para seleccionar) (<span>)
- Fragmento de Texto con el Título de un Trabajo Creativo: Obras, Libros... (<cite>)
Para Modificación de Significado:
- Superíndice (242) (<sup>)
- Subíndice (242) (<sup>)
- Anotaciones menores pequeñas / puntuales (<small>)
- Cita o Frase Extraída de Otro Contexto (<q>)
- Definición (<dfn>)
- Abreviatura o Acrónimo (<abbr>)
- Salto de Línea (<br>)
- Oportunidad de Salto de Línea (división silábica con guión) (<wbr>)
Para Textos Orientados a Aspectos Informáticos:
-Entrada de información del usuario (<kbd>)
- Salida de información de un programa informático (<samp>)
- Variable (<var>)
- Indica una fecha/hora legible para humanos, con formato para máquinas opcional (<time>)
- Información equivalente orientado a máquinas (<data>)
- Fragmento de Código Fuente (<code>)
Para Textos de Edición:
- Fragmento de texto o información añadida a posteriori de la publicación inicial (<ins>)
- Fragmento de texto o información eliminada a posteriori de la publicación inicial (<del>)
Multimedia
- Image (<img>)
- Picture (<pic>)
- Frame (<iframe>)
- Embed (<embed>)
- Object (<object>)
- Parameter (<param>)
- Video (<video>)
- Audio (<audio>)
- Source (<source>)
- Track (<track>)
- Map (<map>)
- Area (<area>)
Tablas HTML
Para Tablas Básicas:
- Etiqueta que Contendrá Toda la Tabla (<table>)
- Table Row (<tr>)
- Table Data (<td>)
- Table Header (<th>)
Para Organización de Tablas:
- Etiqueta que Contendrá la Cabecera de la Tabla (<thead>)
- Etiqueta que Contendrá el Cuerpo de la Tabla (<tbody>)
- Etiqueta que Contendrá el Pie de la Tabla (<tfoot>)
- Establece un Título a la Tabla, independientemente de su posición (<caption>)
Para Agrupar las Columnas de la Tabla:
- Etiqueta Contenedora de Columnas (crea una agrupación de columnas) (<colgroup>)
- Etiqueta que Representa a una Columna de la Tabla (<col>)
Formularios
- Para crear el formulario (<form>)
- Para poner una etiqueta (<label>)
- Para Input (<input>)
- Para Botón (<button>)
- Para Selección (<select>)
- Datalist (<datalist>)
- Optgroup (<optgroup>)
- Option (<option>)
- Text Area (<textarea>)
- Progress (<progress>)
- Meter (<meter>)
- Fieldset (<fieldset>)
- Legend (<legend>)
Scripting
- Script (<script>)
- No Script (<noscript>)
- Template (<template>)
- Canvas (<canvas>)
Sección
- Artículo (<article>)
- Sección (<section>)
- Nav (<nav>)
- Aside (<aside>)
- Address (<address>)
- h1, h2, h3, h4, h5, h6 (<hX>)
- Cabecera (<header>)
- Pie (<footer>)
Ideogramas
- Ruby (<rub>)
- Rt (<rt>)
- Rb (<rb>)
- Rtc (<rtc>)
- Rp (<rp>)
- Bdi (<bdi>)
- Bdo (<bdo>)
Edición
- Insertar (<ins>)
- Eliminar (<del>)
